# Droid Toolbox - M5Stack Cardputer Edition

A port of the original Droid Toolbox firmware for the M5Stack Cardputer.
Control and interact with Galaxy's Edge droids via Bluetooth LE.

## Hardware Required
- M5Stack Cardputer
- Galaxy's Edge Droid (from Droid Depot)
- USB-C cable for flashing

## Features
- Scan for nearby Galaxy's Edge droids
- Connect to droids via BLE
- Trigger audio tracks and control volume
- Emit location and droid beacons
- Rotating and random beacon modes
- Expert beacon mode for full control

## Keybindings
| Key | Action |
|-----|--------|
| Enter (tap) | Select / Confirm |
| Enter (hold) | Back |
| ; | Navigate up |
| , | Navigate down |

## Building
This project uses PlatformIO. Open in VS Code with PlatformIO installed and build/upload directly.
